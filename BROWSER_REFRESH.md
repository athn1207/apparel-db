# ブラウザでサイトを表示・更新して確認する方法

data.json を直したあと、「本当に表示が直ったか」を確認する手順です。  
**今、どこでサイトを見ているか**（自分のパソコンだけか、インターネットのURLか）でやり方が違います。

---

## 「データの読み込みに失敗しました」が出るとき

**index.html をダブルクリックして開いていませんか？**  
アドレスバーが `file:///C:/Users/...` のように **file://** で始まっていると、ブラウザの仕様で **data.json を読み込めず、必ず失敗します。**

**対処：次のどちらかで開いてください。**

1. **Live Server で開く**  
   Cursor で **index.html** を右クリック → **「Open with Live Server」**。  
   アドレスが `http://127.0.0.1:5500/...` のようになります。

2. **GitHub Pages で開く**  
   `git push` したあと、ブラウザで **https://athn1207.github.io/apparel-db/** を開く。

3. **簡易サーバーで開く（Python が入っている場合）**  
   ターミナルでプロジェクトのフォルダに移動してから `python -m http.server 8000` を実行し、ブラウザで **http://localhost:8000** を開く。

## A. ローカル（Live Server）で見ている場合

「Cursor の Live Server で開いたブラウザ」で見ているときの確認方法です。

### 1. サイトのタブを開いたままにする

- すでに **Apparel DB** のページが開いているブラウザのタブをそのままにします。

### 2. 強制再読み込みをする

- そのタブを**クリックして前面に出す**。
- キーボードで次のキーを**同時に**押します。  
  **Ctrl + Shift + R**  
  （Mac の場合は **Cmd + Shift + R**）
- これで「キャッシュを無視して、もう一度サーバーから読み直す」ことができます。

### 3. 表示を確認する

- ページが真っ白になったり、「データの読み込みに失敗しました」が出ていなければ成功です。
- ブランド一覧（AURALEE, NIKE, その他, JACOB COHEN など）が表示されていれば、data.json の修正が反映されています。

### もし Live Server を止めてしまったら

- Cursor で **index.html** を右クリック → **「Open with Live Server」** で再度開く。
- 開いたら、上と同じように **Ctrl + Shift + R** で再読み込みする。

## B. GitHub Pages（インターネットのURL）で見ている場合

**https://athn1207.github.io/apparel-db/** で見ているときは、**Git でプッシュしないと修正は反映されません**。

### 1. 修正を GitHub に送る（プッシュ）

ターミナルで、プロジェクトのフォルダに移動してから、次の3つを実行します。

```powershell
cd "c:\Users\akiko\OneDrive\デスクトップ\apparel-db"
git add data/data.json
git commit -m "data.json を修正"
git push
```

- これで、直した **data.json** が GitHub に送られます。

### 2. 1〜2分待つ

- GitHub Pages が更新されるまで、**1〜2分**かかることがあります。
- その間は「まだ古い内容のまま」でも問題ありません。

### 3. ブラウザで開き直す

- ブラウザの**アドレスバー**に次のURLを入力するか、ブックマークから開きます。  
  **https://athn1207.github.io/apparel-db/**
- そのあと **Ctrl + Shift + R**（Mac は **Cmd + Shift + R**）で**強制再読み込み**します。
- キャッシュで古いページが出ることがあるので、「開き直す ＋ 強制再読み込み」の両方をすると安心です。

### 4. 表示を確認する

- ブランド一覧が表示され、「データの読み込みに失敗しました」が出ていなければ、GitHub Pages にも修正が反映されています。

## まとめ
